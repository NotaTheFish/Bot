# 🎉 Giveaway Bot — Бот для розыгрышей в Telegram

## Установка

### 1. Клонируй/скопируй файлы проекта

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой .env
Скопируй `.env.example` в `.env` и заполни:
```
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=твой_числовой_ID_в_Telegram
```

Свой ID можно узнать у бота @userinfobot

### 4. Запусти бота
```bash
python bot.py
```

---

## Использование

### Создание конкурса (в личке бота)
1. Нажми **➕ Создать конкурс**
2. Введи название, ключ, текст объявления
3. Укажи количество призовых мест и описание каждого приза
4. Добавь каналы/чаты для проверки подписки (перешли сообщение или введи ID)
5. Напиши `готово` когда все каналы добавлены

### Добавление бота в каналы
- Добавь бота в канал/чат как администратора
- Минимальные права: **Добавление участников** (для проверки подписки)

### Запуск конкурса
**В чате:** напиши `!ключ!` — бот опубликует объявление
**В канале:** выбери конкурс → **📢 Запустить в канале** → введи ID канала

### Завершение
- **✅ Завершить конкурс** — случайный выбор победителей по местам
- **❌ Отменить конкурс** — досрочное завершение без победителей
