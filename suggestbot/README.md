# SuggestBot — бот предложки для новостного канала

## Структура

```
suggestbot/
├── bot.py                  # точка входа
├── config.py               # переменные окружения
├── states.py               # FSM состояния
├── requirements.txt
├── .env.example
├── Procfile                # Railway
├── db/
│   ├── database.py         # пул соединений + init таблиц
│   └── queries.py          # все SQL запросы
├── handlers/
│   ├── common.py           # /start
│   ├── user.py             # флоу предложки
│   └── admin.py            # всё для админов
├── keyboards/
│   ├── user_kb.py
│   └── admin_kb.py
├── middlewares/
│   ├── ban_check.py        # проверка бана + upsert юзера
│   └── antiflood.py        # Redis антифлуд с обратным отсчётом
└── utils/
    ├── helpers.py           # вспомогательные функции
    ├── logger.py            # логирование в чат
    └── media_group.py      # буфер медиагрупп в Redis
```

## Переменные окружения

Скопируй `.env.example` в `.env` и заполни:

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `ADMIN_IDS` | ID админов через запятую: `123,456,789` |
| `CHANNEL_ID` | ID канала: `-1001234567890` |
| `LOG_CHAT_ID` | ID чата для логов (можно отдельный чат или тот же канал) |
| `DATABASE_URL` | PostgreSQL URL от Railway |
| `REDIS_URL` | Redis URL от Railway |

## Деплой на Railway

1. Создай новый проект в Railway
2. Добавь сервисы: **PostgreSQL** и **Redis**
3. Создай сервис из GitHub репозитория
4. В Variables добавь все переменные из таблицы выше
5. Railway подхватит `Procfile` автоматически

## Первый запуск

1. Запусти бота
2. Напиши `/start` — получишь меню с кнопкой **➕ Добавить канал**
3. Добавь бота в канал как администратора с правом **публикации**
4. Перешли любое сообщение из канала боту — он запомнит channel ID
5. Пропиши этот ID в `CHANNEL_ID` и перезапусти

## Таблицы БД

Все таблицы с префиксом `sgb_` создаются автоматически при старте:

- `sgb_users` — пользователи (бан/разбан)
- `sgb_submissions` — заявки со статусами `pending / accepted / rejected`
- `sgb_submission_content` — контент заявок (фото/видео/текст)
- `sgb_settings` — настройки бота (`forward_mode`, `show_author_btn`)

## Флоу юзера

```
/start → [Предложить новость или работу]
       → бот просит контент
       → юзер шлёт текст / фото / видео (или всё вместе)
       → превью + [Отправить] [Отменить]
       → заявка идёт всем админам
       → антифлуд 5 минут
```

## Флоу админа

```
Входящая заявка:
  [👤 Перейти в личку] [✅ Принять] [❌ Отклонить] [💬 Ответить] [🚫 Забанить]

Принять:
  → [🚀 Публикую как есть] или [✏️ Редактирую]
  → [👤 Добавить кнопку автора?] Да / Нет
  → публикация в канал
  → уведомление юзеру ✅

Отклонить → заявка скрыта, юзер не уведомляется

Меню:
  📋 Все актуальные заявки — листалка с навигацией
  ⚙️ Настройки — forward_mode, кнопка автора
  🚫 Забаненные — список + кнопки разбана
  ➕ Добавить канал — одноразовая настройка
```
